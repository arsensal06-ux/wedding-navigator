# 💍 Свадебный Навигатор

Интерактивный веб-сайт для гостей свадьбы Александры и Дмитрия с программой, рассадкой, меню и настройками для автора.

## 🌐 Открыть сайт

**[arsensal06-ux.github.io/wedding-navigator](https://arsensal06-ux.github.io/wedding-navigator)**

## ✨ Возможности

### 👤 Для гостей:
- 📱 Выбор профиля гостя с личной информацией
- ⏳ Обратный отсчет до свадьбы
- 🎯 Таймлайн мероприятия с текущим событием
- 🪑 План рассадки со своим столом
- 🍽️ Меню с учетом а��лергий
- 📱 QR-код для быстрого доступа

### ⚙️ Для автора:
- 🔐 Защита паролем (2310)
- 📝 Редактирование имен молодожёнов
- 📅 Изменение даты и времени события
- 🕐 Настройка времени и описания каждого события
- 💾 Автосохранение всех изменений

## 🔐 Доступ к настройкам

**Пароль для входа в панель автора:** `2310`

## 📱 Совместимость

- ✅ Мобильные устройства (iOS, Android)
- ✅ Планшеты
- ✅ Десктопные браузеры
- ✅ Offline режим (данные сохраняются локально)

## 🔧 Технологии

- HTML5
- CSS3 (адаптивный дизайн)
- JavaScript (vanilla, без зависимостей)
- QR Code Library (QRCode.js)
- GitHub Pages (хостинг)

## 💾 Данные

Все изменения сохраняются в localStorage браузера и не требуют сервера.
